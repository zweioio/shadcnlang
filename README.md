# shadcn/ui 本地化翻译扩展

将 [shadcn/ui](https://ui.shadcn.com/) 官方文档网站翻译为您偏好的语言。

## 与其他翻译工具的区别

| 特性 | 本扩展 | 普通翻译插件（Google 翻译等） |
|------|--------|------------------------------|
| **翻译方式** | 预编译字典查询 | 实时调用翻译 API |
| **网络依赖** | 完全离线，零网络请求 | 每次翻译都需要 API 调用 |
| **翻译质量** | 人工审校，术语准确 | 机器翻译，技术术语可能错误 |
| **速度** | 即时，毫秒级完成 | 需等待 API 响应 |
| **代码保护** | 精确跳过代码块 | 可能误翻译代码内容 |
| **隐私** | 数据不离开本地 | 文本发送到第三方服务器 |

## 工作原理

本扩展采用 **预翻译字典** 方案：
1. 从 shadcn/ui 网站提取所有可见英文文本
2. 由 AI 精校生成目标语言的高质量翻译
3. 翻译字典打包在扩展中（`dict/` 目录）
4. 运行时直接在本地查表替换 — **完全不需要网络请求**

## 安装方法

### 开发模式安装（Chrome）

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `shadcn-translator` 文件夹

### 使用

1. 访问 [https://ui.shadcn.com/](https://ui.shadcn.com/) 任意页面
2. 点击工具栏中的扩展图标
3. 选择目标语言
4. 点击 **"立即翻译"** 按钮，或开启 **"自动翻译"**

```

├── manifest.json            # 扩展清单 (V3)
├── popup/                   # 弹窗界面
│   ├── popup.html           # 弹窗 HTML
│   ├── popup.css            # 弹窗样式
│   └── popup.js             # 弹窗逻辑

│   └── content.css          # 翻译指示器样式
```

## 如何添加新语言

1. 在 `dict/` 目录下创建 `{语言代码}.json` 文件
2. 按照 `zh-CN.json` 的结构填写翻译条目
3. 在 `popup/popup.js` 的 `SUPPORTED_LANGUAGES` 中添加语言
4. 重新加载扩展即可
