# shadcn/ui 中文翻译扩展

将 [shadcn/ui](https://ui.shadcn.com/) 官方文档自动翻译为简体中文的 Chrome 扩展。**安装即用，无需任何手动操作。**

## ✨ 特性

- **自动翻译** — 安装扩展后访问 `ui.shadcn.com` 自动翻译为中文，无需点击任何按钮
- **零网络请求** — 全部采用本地预编译字典，毫秒级完成翻译，离线可用
- **术语准确** — 技术词汇人工审校，比机器翻译更准确
- **导航栏切换** — 页面顶部导航栏有 🌐 语言图标，点击即可切换中/英文
- **深色模式适配** — 图标颜色自动跟随网站主题色
- **海量字典** — 内置 **2000+** 条翻译条目，覆盖全部文档页面

## 📦 安装

### 开发模式安装

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `shadcn-translator` 文件夹

### 使用

1. 访问 [https://ui.shadcn.com/](https://ui.shadcn.com/) 任意页面
2. 页面自动翻译为中文
3. 点击导航栏的 🌐 图标切换中/英文

## 🔧 技术原理

采用**预翻译字典**方案：

1. 从 shadcn/ui 源码提取全部可见英文文本
2. 生成目标语言的高质量翻译
3. 翻译字典打包在扩展中（`dict/zh-CN.json`）
4. 运行时 content script 遍历 DOM 查表替换

```
DOM 加载 → 收集可见文本节点 → 字典查找 → 原地替换 → MutationObserver 监听动态内容
```


## 🔄 工作流程

```
┌─────────────────────────────────────────┐
│  用户访问 ui.shadcn.com                  │
└────────────────┬────────────────────────┘
                 ↓
┌─────────────────────────────────────────┐
│  content script 自动注入                  │
│  ├── 设置导航监听 (SPA 页面切换自动翻译)   │
│  ├── 在导航栏插入 🌐 语言切换图标          │
│  └── 100ms 后自动执行首次翻译             │
└────────────────┬────────────────────────┘
                 ↓
┌─────────────────────────────────────────┐
│  翻译引擎                                │
│  ├── 遍历 DOM + Shadow DOM + Iframe     │
│  ├── 字典查找匹配                        │
│  └── 原地替换文本，不破坏页面结构          │
└─────────────────────────────────────────┘
    ↓                          ↑
  DOM 变化 ── MutationObserver ── 自动重新翻译
```
